# Module Index

This index maps the frozen modules of the Human-in-the-Loop (HITL) packages:

- **`app.platform.hitl`**: Package init, imports, and exports.
- **`app.platform.hitl.engine`**: Approval engines, reviewer assignments, and policy evaluations.
- **`app.platform.hitl.intervention`**: Execution controllers, pause, resume, cancel, and restart managers.
- **`app.platform.hitl.notifications`**: Email/Slack agnostic delivery, reminder schedules, and group timeout escalations.
- **`app.platform.hitl.models`**: Pydantic baseline schemas and models.
