# Release Notes v3.0.0-rc1

We are pleased to announce the Release Candidate preparation of the AI Platform version `v3.0.0-rc1`.

## Key Features in this release
- **Human-in-the-Loop Orchestration**: Native support for pauses, continuation validations, consensus policy approvals, and group escalations.
- **Strict Checkpointing Persistence**: Integrated checkpoint recovery adapter boundaries to assure clean rollback RPO bounds.
- **Zero-Bypass Architecture**: Enforced decoupling and provider-agnostic interfaces.
